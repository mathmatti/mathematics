1 Limit point of a set

1.1 Definition

Let X be a metric space. All points and sets mentioned below are understood to be elements and subsets of X.

A point p is a limit point of the set E if every neighborhood of p contains a point q≠p such that q ∈ E.

1.2 Remarks

Scriviamo la definizione di neighborhood in maniera estesa:

Nr(p) = {q ∈ X : d(p,q) < r,r > 0}

La definizione dice che ogni Nr(p) contiene un q≠p

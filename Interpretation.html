2 DEFINIZIONE (2)

An interpretation I is a function which assigns to any atomic formula pi a truth value

I(pi) ∈ 0,1

If I(pi) = 1 then pi is called true under the interpretation I If I(pi) = 0 then pi is called false under the interpretation I

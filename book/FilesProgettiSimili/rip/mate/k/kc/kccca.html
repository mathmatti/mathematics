Giudizio particolare affermativo

Mediante la teoria degli insiemi, e' possibile mostrare la verita' o meno del giudizio particolare affermativo:
Partiamo dal nostro giudizio:
Esiste un x tale che x e' un uomo ed x ha 90 anni

Considero gli insiemi

P(x) = { x : x e' un uomo }
Q(x) = { x : x ha 90 anni }

la scrittura

x, P(x) Q(x)

implica che l'intersezione fra gli insiemi P(x) e Q(x) non e' vuota
P(x) Q(x) Ø
Nella figura la parte gialla e' la parte comune (gli uomini novantenni)
la congiunzione logica corrisponde all'intersezione in teoria degli insiemi

Potenza di frazioni algebriche
Ripassiamo l'operazione di elevamento a potenza per frazioni numeriche e poi facciamo l'equivalente con i polinomi
se ad esempio ho:
( 3 / 5 ) 3 =
devo fare la potenza sia al numeratore che al denominatore
  3 3       27
----- = ------
 5 3      125
Quando ho una potenza di frazioni algebriche devo fare la stessa cosa:
Elevare a potenza sia il numeratore che il denominatore
Esempio: calcolare la seguente potenza:

[ (x+2)2 / (x2 - 9) ]2 =

   ( x + 2 )4
= ----------- =
   ( x2 - 9)2

          (x + 2)4
= -------------------
   (x + 3)2·(x - 3)2

Se si vuole ora si possono calcolare i prodotti notevoli ma di solito e' preferibile non farlo

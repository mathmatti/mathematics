sommo le due frazioni al numeratore
          y                               1
  -------------------------- + ------------------- =
(x - y)·(x2 + xy + y2)       x2 + xy + y2

il minimo comune multiplo e'

(x - y)·(x2 + xy + y2)

quindi ottengo:

     y + 1·(x - y)
= ------------------------- =
(x - y)·(x2 + xy + y2)

eseguo la moltiplicazione al numeratore (il denominatore essendo scomposto non va moltiplicato)

     y + x - y
= ----------------------- =
(x - y)·(x2 + xy + y2)

sommo i termini simili: y va via con -y ed ottengo

          x
= ----------------------------
   (x - y)·(x2 + xy + y2)

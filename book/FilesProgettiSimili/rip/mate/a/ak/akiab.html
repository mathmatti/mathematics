Risolvere la disequazione

  x - 3     x2 -3x +2
  • Il radicando deve sempre essere maggiore o uguale a zero
    x2 - 3x + 2 0
  • Essendo il radicale definito positivo o nullo anche il primo termine (essendo maggiore o uguale al secondo) dovra' essere positivo o nullo
    x - 3 0
  • Il quadrato del primo termine dovra' essere maggiore o uguale al quadrato del secondo termine
    (x - 3)2 x2 - 3x + 2
Debbo quindi risovere il sistema
x2 - 3x + 2 0
x - 3 0
(x-3)2 x2 - 3x + 2

dopo alcuni calcoli otteniamo:

x2 - 3x + 2 0

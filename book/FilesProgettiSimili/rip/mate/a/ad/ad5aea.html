Esercizio
(x6 - 64):(x2 -4)=
Prima di impostare la divisione dobbiamo rendere i polinomi ordinati;
Possiamo considerarlo come
(x6 + 0x5+ 0x4 + 0x3 + 0x2 + 0x - 64):(x2 + 0x -4)=
Ora impostiamo la divisione

 x6    //    //    //    //    //    -64   x2   //  -4            
          


Essendo il divisore composto da tre spazi consideriamo tre spazi
facciamo il primo termine diviso il primo termine  x6 : x2 = x4
scrivo x4al primo posto nel divisore

 x6    //    //    //    //    //   -64   x2   //  -4            
         x4 


ora moltiplico x4 per il divisore x2 -4 ed ottengo x6 - 4x4
Naturalmente devo considerarlo ordinato cioe'
x6 +0x5 - 4x4
ora lo scrivo nelle rispettive colonne cambiandolo di segno

 x6    //    //    //    //    //    -64   x2   //  -4            
-x6  //  +4x4          x4 


sommo in verticale ed ottengo

 x6    //    //    //    //    //    -64   x2   //  -4            
-x6  //  +4x4          x4   
 //   //   +4x2      


ora abbasso due posti perche' devo sempre considerarne tre (tanti quanti sono i posti del divisore)

-x6  //  +4x4          x4   
 //   //   +4x4  //   //

4x4 : x2 = +4x2
Scrivo +4x2 al secondo posto del quoziente

-x6  //  +4x4          x4   +4x2

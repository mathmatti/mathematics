Caso del discriminante maggiore di zero
Facciamo un esempio
x2 - 3x - 4= 0
i coefficienti sono
  • a = 1
  • b = - 3
  • c = -4
 sostituisco nella formula
         -(-3) [(-3)2 - 4(1)(-4)]
x1,2 = ----------------------------------
                 2(1)
eseguo i calcoli
         3 (9 + 16)
x1,2 = -------------------------
                 2
il discriminante vale 25 > 0
         3 25
x1,2 = -----------------
                 2
estraggo di radice. Posso farlo perche' il discriminante e' maggiore di zero
         3 5
x1,2 = ------------
            2
ora devo fare un bivio scegliendo una volta il meno e l'altra il piu'
  • x2 = (3 - 5)/2 = -2/2 = -1
  • x1 = (3 + 5)/2 = 8/2 = 4

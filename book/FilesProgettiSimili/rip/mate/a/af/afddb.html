Risolviamo gli stessi esercizi della pagina precedente
primo esempio
2x4 = 32
 divido per 2
x4 = 16
 x4 - 16 = 0
 scompongo
x4 - 16 = (x2 - 4)(x2 + 4) = 0
Pongo uguali a zero i fattori
  • x2 - 4 = 0
  • x2 + 4 = 0
    Risolvo la prima
    x = 4
  • x1 = - 2    x2 = + 2
    Risolvo la seconda
    x = (-4)
  • x1 = - 2i     x2 = + 2i

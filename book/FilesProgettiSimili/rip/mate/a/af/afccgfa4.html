data l'equazione
(k+1)x2 -2kx + k - 2 = 0
trovare il valore di k affinche' la somma degli inversi delle radici valga 3

1 1
---- + ---- = 3
x1 x2

x1 + x2
------------------ = 3
x1 x2
sostituisco alla somma -b/a ed al prodotto c/a ed ottengo
- b/a
--------- = 3
c/a

b a
- ---- · ---- = 3
a c
quindi semplificando
- b
----- = 3
c
abbiamo
a = k+1
b = -2k
c = k - 2
quindi sostituendo
- b -(-2k)
----- =------- = 3
c k-2
devo risolvere l'equazione
2k
-------- = 3
k - 2
Per poterla risolvere dobbiamo porre la condizione di realta'
C.R.      k-2 0       k 2
ora posso fare il m.c.m. e togliere il denominatore (e' piu' semplice moltiplicare in croce)
2k = 3(k -2)
2k = 3k - 6
2k - 3k = -6
-k = -6

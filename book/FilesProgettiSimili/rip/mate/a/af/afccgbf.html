risolvere la seguente equazione:
ax2 + ax = 2x(x - 1)

E' un' equazione letterale, prima eseguiamo la moltiplicazione


ax2 + ax = 2x2 - 2x

portiamo tutto prima dell'uguale

ax2 + ax - 2x2 + 2x = 0

Sommo i termini simili
quando ho delle lettere e dei numeri per indicare che sommo faccio il raccoglimento a fattor comune
x2(a - 2) + x(a + 2) = 0

raccolgo a fattor comune x

x[x(a-2) + (a+2)] = 0

x =0
x(a-2) + (a+2) = 0

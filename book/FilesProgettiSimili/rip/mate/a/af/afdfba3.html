Esercizio

Risolvere:
x + 2 + (x + 2) = 0
Isolo la radice portando gli altri termini dopo l'uguale
(x + 2) = - x - 2
Elevo a quadrato da una parte e dall'altra
[(x + 2)]2 = (- x - 2)2
Eseguo il quadrato dopo l'uguale. Prima dell'uguale semplifico il quadrato con la radice
x + 2 = x2 + 4x + 4
Trasporto tutto prima dell'uguale
x + 2 - x2 - 4x - 4 = 0
Sommo
-x2 - 3x - 2 = 0
Cambio segno
x2 + 3x + 2 = 0
Risolvo
         -b (b2 - 4ac)
x1,2 = ----------------------
                  2a

         -3 (9 - 8)
x1,2 = ----------------------
                  2

         -3 1
x1,2 = ---------
            2

Le radici sono:
x1 = (-3 - 1)/2 = -2
x2 = (-3 + 1)/2 = -1
Non e' finita! Devo fare la verifica
  • Verifica soluzione x = -2
    Sostituisco nell'equazione di partenza -2 al posto di x
    -2 + 2 + (-2 + 2) = 0
    0 = 0
    L'uguaglianza e' valida e la soluzione x=-2 e' accettabile
  • Verifica soluzione x = -1
    Sostituisco nell'equazione di partenza -1 al posto di x
    -1 + 2 + (-1 + 2) = 0
    1 + 1 = 0
    2 = 0
    L'uguaglianza non e' valida e la soluzione x=-1 non e' accettabile
Una sola soluzione x = -2

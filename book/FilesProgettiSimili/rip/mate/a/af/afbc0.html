In matematica non si puo' mai dividere per zero
Infatti essendo la divisione il contrario della moltiplicazione se io avessi ad esempio
5 : 0 = 2
questo equivale a
5 = 2 x 0 = 0
se invece fosse
5 : 0 = 4
questo equivale a
5 = 4 x 0 = 0
vedi che non riesco ad avere un risultato ben determinato.
allora in matematica il dividere per zero porta a risultati assurdi

Equazioni biquadratiche

Si chiamera' biquadratica un'equazione che ha tre termini, uno con x4, uno con x2, ed un termine noto.
ax4 + bx2 + c = 0
 Per risolvere quest'equazione conviene sostituire x2 con y e quindi x4 con y2, quindi l'equazione diviene di secondo grado in y;
ay2 + by + c = 0
 trovate le due soluzioni y1 ed y2 dovro' risolvere le due equazioni
x2 = y1
x2 = y2
Le 4 soluzioni trovate saranno le soluzioni dell'equazione di partenza.
In qualche libro di testo viene addirittura fornita una formula finale
-b (b2 - 4ac)
----------------------
2a

x4 - 5x2 + 4 = 0
 pongo
x2 =y e quindi x4 = y2
y2 - 5y + 4 = 0

  • y1 = (5 + 3)/2 = 8/2 = 4

Ora devo risolvere le due equazioni:
  • x2 = 4
  • x2 = 1

    • risolvo la prima
    • x1 = -2
    • x2 = +2

    • x3 = -1
    • x4 = +1
quindi le 4 soluzioni (messe in fila) sono:
x1 = -2   x2 = -1   x3 = +1   x4 = +2

Equazione di secondo grado con riferimento alla somma e prodotto dei coefficienti


Alla luce di quanto visto nelle pagine precedenti possiamo scrivere l'equazione di secondo grado anche come
x2 - sx + p = 0

con
s somma delle radici
p prodotto delle radici
infatti tu hai l'equazione
ax2 + bx + c = 0
divido tutto per a
x2 + b
----
a 		x + c
----
a 		= 0
siccome b/a e' la somma delle radici cambiata di segno, mentre c/a e' il prodotto posso scrivere
b
----
a 		= - s           c
----
a 		= p
e sostituendo ottengo
x2 - sx + p = 0

dimostriamo la formula
x3 + y3 = (x + y)3 - 3xy(x+y)

Partiamo dall'uguaglianza (cubo del binomio)

(x + y)3 = x3 + 3x2y + 3xy2 + y3

leggiamo a rovescio

x3 + 3x2y + 3xy2 + y3 = (x + y)3

raccolgo 3xy fra il secondo ed il terzo termine al primo membro

x3 + 3xy(x+y) + y3 = (x + y)3

porto il 3xy(x+y) dopol'uguale

x3 + y3 = (x + y)3 - 3xy(x+y)

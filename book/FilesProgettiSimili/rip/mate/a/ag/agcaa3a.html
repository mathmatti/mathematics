x2 + 2x - 3 = 0
Usando la formula completa:
         -2 [(-2)2 - 4·1·(-3)]
x1,2 = --------------------------
                  2

         -2 (4 + 12)
x1,2 = ------------------
                2

         -2 16

                2

         -2 4

            2

x1 = 1
x2 = -3
oppure usando la formula ridotta:
x2 + 2x - 3 = 0

                  1

x1,2 = -1 4

x1,2 = -1 2

x1 = 1

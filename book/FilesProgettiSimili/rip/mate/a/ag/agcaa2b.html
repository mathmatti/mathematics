Per calcolare il segno dell'espressione
(x+2) (x-3) < 0
devi fare il prodotto dei segni di (x+2) e di (x-3)
li pongo entrambe maggiori di zero
Porre (x+2) > 0 equivale a dire x > -2
Porre (x-3) > 0 equivale a dire x > 3
x > -2                  - - - - - - - (-2)+ + + + + + + + + + + + + + + + +
x > 3                  - - - - - - - - - - - - - - - - (3) + + + + + + + +
f(x)                     + + + + + (-2) - - - - - - (3) + + + + + + + + +

per x < -2 abbiamo x + 2 negativa ed x - 3 negativa quindi il loro prodotto fx) = (x + 2)(x - 3) e' positivo (meno per meno fa piu')
per -2 < x < 3 abbiamo x + 2 positiva ed x - 3 negativa quindi il loro prodotto e' negativo (piu' per meno da' meno)
per x > 3 abbiamo x + 2 positiva ed x - 3 positiva quindi il loro prodotto e' positivo (piu' per piu' fa piu')
 Siccome devo cercare dove il prodotto f(x)= (x + 2)(x - 3) risulta minore di zero allora la soluzione sara'    -2 < x < 3

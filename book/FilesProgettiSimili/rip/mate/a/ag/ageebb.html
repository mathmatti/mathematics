abbiamo la disequazione:
x2 - 5x > 0
Considero l'equazione associata
x2 - 5x = 0
raccolgo la x
x(x - 5) = 0

Ottengo le soluzioni

x1 = 0                   x2 = 5
Siccome le due radici sono reali e distinte (e quindi maggiore di zero) allora la mia disequazione sara' valida per valori esterni all'intervallo delle radici, cioe'
x < 0 U x > 5

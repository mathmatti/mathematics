Algebra delle matrici


Riassumendo le proprieta' viste:
  • Esiste un'operazione interna: la somma
  • La somma e' commutativa
  • esiste l'elemento neutro Ø rispetto all'operazione somma
  • Data una matrice A esiste l'elemento opposto -A tale che A (-A) = Ø
  • Esiste un prodotto esterno x (quindi con queste proprieta' l'insieme delle matrici quadrate e' uno spazio vettoriale)
  • Esiste un'operazione interna: il prodotto righe per colonne
  • Esiste l'elemento neutro moltiplicativo u
  • Per ogni matrice quadrata non singolare esiste l'elemento inverso tale che
    A A-1 = A-1 A = u
  • Il prodotto e' associativo
    (A B) C = A (B C)
  • la moltiplicazione e' distributiva rispetto all'addizione
    A ( B C) = (A B) (A C)
    Proprieta' non verificata, prova a verificarla tu per esercizio
Quindi si dice che l'insieme delle matrici quadrate di ordine nxn forma un'algebra

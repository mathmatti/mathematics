dimostriamo che l'equazione dell'iperbole equilatera
x2 - y2 = a2
con una rotazione di 45° si trasforma nell'equazione dell'iperbole equilatera riferita ai propri assi
XY = K
consideriamo le formule di trasformazione (rotazione di 45°)
x = X 2 /2 + Y 2 /2
y = - X 2 /2 + Y 2 /2
andiamo a sostituire nell'equazione dell'iperbole

(X 2 /2 + Y 2 /2)2 - ( - X2 /2 + Y 2 /2)2 = a2

sviluppo i quadrati ricordando che (2 /2)2 = 1/2 ottengo

X2 /2 + XY + Y2/2 - (X2 /2 - XY + Y2 /2) = a2

tolgo le parentesi

X2 /2 + XY + Y2/2 - X2 /2 + XY - Y2 /2 = a2

sommo i termini simili: ottengo

2XY = a2

XY = a2/2

Se ora pongo a2/2 = K (Posso farlo perche' a e' una costante)
ottengo la formula finale
XY = K

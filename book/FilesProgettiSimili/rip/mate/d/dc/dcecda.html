Dimostrazione dell'equazione segmentaria della retta

Troviamo la formula dell'equazione segmentaria della retta
x y
---   + --- = 1
p q

Ho i punti

P=(p,0)       Q=(0,q)

Applico la formula per trovare la retta passante per due punti:
y - y1 x - x1
----------- = -----------   
y2 - y1 x2 - x1

si ha:

x1 = p     y1 = 0     x2 = 0     y2 = q

sostituendo ottengo
y - 0 x - p
----------- = -----------   
q - 0 0 - p

quindi

y x - p
------ = ------   
q - p

moltiplico in croce (che equivale a fare il m.c.m.)

-py = qx - pq

-py - qx = - pq

cambio di segno

qx + py = pq

divido tutti i termini per pq

qx py pq
------ + ------ = ------
pq pq pq

semplificando termine a termine ottengo la formula finale
x y
---   + --- = 1
p q

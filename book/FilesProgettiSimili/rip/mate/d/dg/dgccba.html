Condizione di passaggio per un punto

Per scrivere la condizione di passaggio della parabola
y = ax2 + bx + c
per il punto P = ( x0, y0)
basta sostituire alle variabili x, y nell'equazione le coordinate x0, y0, cioe'
y0 = ax02 + bx0 + c
Esempio scrivere la condizione di passaggio della parabola
y = ax2 + bx + c
per il punto P = ( 2, 3 )
sostituisco a x il valore 2 ed a y il valore 3
3 = a·22 + b·2 + c
3 = 4a + 2b + c
quindi la condizione richiesta e'
4a + 2b + c = 3

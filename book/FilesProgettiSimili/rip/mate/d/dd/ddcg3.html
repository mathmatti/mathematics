Tangenza ad una retta

Imporre la condizione di tangenza ad una retta significa fare il sistema fra la retta e la circonferenza e porre il Delta uguale a zero
Esempio
Trovare la condizione perche' la circonferenza
x2+ y2 + ax + by + c = 0
sia tangente alla retta
y = 2x + 3
Faccio il sistema
x2+ y2 + ax + by + c = 0
y = 2x + 3
 Equazione risolvente
x2+ (2x+3)2 + ax + b(2x+3) + c = 0
x2+ 4x2 + 12x + 9 + ax + 2bx + 3b + c = 0
5x2 + x(a + 2b + 12) + 3b + c + 9 = 0
Pongo il Delta = 0
= b2 - 4ac = 0
 Condizione di tangenza
(a + 2b + 12)2 - 4(5)(3b + c + 9)=0
 Se vuoi vedere la formula del quadrato del trinomio
a2 + 4b2 + 144 + 4ab + 24a + 48b - 60b - 20c - 180 =0
 Condizione di tangenza
a2 + 4b2 + 4ab +24a - 12b - 20c - 36 = 0
Come vedi i calcoli sono piuttosto complicati, quindi viene usata abbastanza raramente, od almeno con rette mancanti di qualche termine

esercizio

xo2 + yo2 - r2 = c
Devo ricavare r, lo isolo portando gli altri termini dopo l'uguale
- r2 = c - xo2 - yo2
Cambio di segno
r2 = xo2 + yo2 - c
Applico la radice da una parte e dall'altra
r2 = (xo2 + yo2 - c)
Nel primo termine tolgo il primo quadrato con la radice
r = (xo2 + yo2 - c)
Talvolta si preferisce la formula equivalente
r = [(a/2)2 + (b/2)2 - c]

Se nella circonferenza mancano i termini ax, by ed il termine noto c

In tal caso a=0   b = 0 e c = 0 e l'equazione della circonferenza diventa
x2+ y2 = 0
E' una circonferenza di centro l'origine e raggio zero (circonferenza degenere)
Si puo' pensare come il prodotto di due rette immaginarie
(x+iy)(x-iy)=0

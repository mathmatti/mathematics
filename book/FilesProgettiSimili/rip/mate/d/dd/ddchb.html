Dimostrazione teorica


Quello che abbiamo fatto come esercizio facciamolo in modo teorico:
Considero le due circonferenza:
x2+ y2 + a1x + b1y + c1 = 0
x2+ y2 + a2x + b2y + c2 = 0
metto a sistema le due equazioni
x2+ y2 + a1x + b1y + c1 = 0
x2+ y2 + a2x + b2y + c2 = 0
sottraggo le due equazioni termine a termine; per non sbagliare prima cambiamo di segno tutti i termini della seconda equazione poi facciamo la somma in verticale
x2+ y2 + a1x        + b1y        + c1 = 0
-x2- y2 - a2x        - b2y         - c2 = 0
----------------------------------------------------------
    //     // (a1 - a2)x + (b1 - b2)y + c1 - c2 = 0
e questa e' un'equazione di primo grado che sostituita ad una delle due equazioni del sistema rende il sistema di secondo grado
(a1 - a2)x + (b1 - b2)y + c1 - c2 = 0
x2+ y2 + a2x + b2y + c2 = 0

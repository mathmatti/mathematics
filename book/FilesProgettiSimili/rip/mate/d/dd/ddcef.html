Se nella circonferenza mancano il termine by ed il termine noto c

In tal caso b = 0 e c = 0 e l'equazione della circonferenza diventa
x2+ y2 + ax = 0
Basta sommare i due fatti:
Siccome manca il termine by la circonferenza ha il centro su un punto dell'asse delle x (equazione y=0), inoltre poiche' manca il termine noto la circonferenza passa per l'origine;
Esempio, considero la circonferenza:
x2+ y2 - 6x = 0
il centro vale
xo = -a/2 = 3
yo = -b/2 = 0
ed il raggio vale
r = (xo2 + yo2 - c) =
= (9 + 0 - 0) = 9 = 3
E' la circonferenza di centro C(3,0) e raggio 3

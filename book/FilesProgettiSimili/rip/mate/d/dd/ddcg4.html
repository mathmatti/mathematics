Valore di una coordinata del centro

Per imporre la condizione del valore di una coordinata del centro basta ricordare le relazioni fra le coordinate del centro ed i coefficienti a e b della circonferenza
a = -2xo
b = -2yo
Esempio
Trovare la condizione perche' la circonferenza
x2+ y2 + ax + by + c = 0
abbia il centro sulla retta
x = 3
significa che il centro ha coordinata x uguale a 3
impongo la condizione
a = -2xo
a = -2(3)
Condizione
a = -6
Altro esempio
Trovare la condizione perche' la circonferenza
x2+ y2 + ax + by + c = 0
abbia il centro sull'asse delle x
l'asse x ha equazione
y = 0
Significa che la coordinata y del centro vale zero
impongo la condizione
b = -2yo
b = -2(0)
Condizione
b = 0

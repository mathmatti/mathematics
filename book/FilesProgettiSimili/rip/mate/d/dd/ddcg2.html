Passaggio per un punto

Imporre la condizione di passaggio per un punto significa prendere l'equazione della circonferenza
x2+ y2 + ax + by + c = 0
e sostituire al posto di x e y le coordinate del punto dato.
Esempio
Condizione per il passaggio della circonferenza
x2+ y2 + ax + by + c = 0
per il punto A(2,3)
(2)2+ (3)2 + a(2) + b(3) + c = 0
4 + 9 + 2a + 3b + c = 0
Condizione
2a + 3b + c + 13 = 0
 Da notare che il passaggio per l'origine
O=(0,0)
porta alla condizione
c = 0
infatti sostituendo le coordinate nell'equazione della circonferenza ottengo:
(0)2+ (0)2 + a(0) + b(0) + c = 0
Condizione
c = 0

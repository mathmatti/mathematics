Indipendenti significa che non li posso ridurre di numero:
ad esempio se considero l'equazione della circonferenza
ex2+ ey2 + fx + gy + h = 0
con e f g h parametri; posso dividere tutti i termini per e ed ottengo
x2+ y2 + (f/e) x + (g/e)y + (h/e) = 0
se pongo
f/e = a
g/e = b
h/e = c
 ho l'equazione
x2+ y2 + ax + by + c = 0
Ho ridotto i parametri da 4 a 3, ma non potrei ridurre ulteriormente

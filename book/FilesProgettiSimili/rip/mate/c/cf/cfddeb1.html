Svolgimento
y = x2 ex + xex
si tratta di una somma fra le due funzioni
x2 ex e xex ed ognuna di esse e' un prodotto di funzioni
quindi faccio la somma delle derivate
  • la derivata di x2 ex e' 2x·ex + x2·ex
    (derivata del prodotto fra x2 ed ex )
  • la derivata di x ex e' ex + x·ex
    (derivata del prodotto fra x ed ex )
Quindi
y' = 2x·ex + x2·ex + ex + x·ex
y' = 3x·ex + x2·ex + ex
raccogliendo ex ed ordinando
y' = ex (x2 + 3x + 1)

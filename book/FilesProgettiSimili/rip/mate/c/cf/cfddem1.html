( 1 + xn ) m
y = --------------
1 - xn
E' la potenza di una frazione: prima deriviamo la potenza e, successivamente la frazione; per mostrartelo meglio fra la derivata della potenza e quella della frazione lascio un po' di spazio
( 1 + xn ) m-1 n xn-1 (1 - xn) - (-n) xn-1 (1 + xn)
y' =   m --------------           ----------------------------------------------------------
1 - xn (1 - xn)2


( 1 + xn ) m-1 n xn-1 (1 - xn) + n xn-1 (1 + xn)
y' =   m --------------           ----------------------------------------------------------
1 - xn (1 - xn)2
Possiamo scriverlo in modo piu' compatto mettendo in evidenza al numeratore n xn-1, ottengo:
( 1 + xn ) m-1 n xn-1 (1 - xn + 1 + xn)
y' =   m --------------           ----------------------------------------------------------
1 - xn (1 - xn)2


( 1 + xn ) m-1 2m n xn-1
y' = --------------     ---------------------------    
1 - xn (1 - xn)2

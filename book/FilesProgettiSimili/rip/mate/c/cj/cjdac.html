  • La derivata di 0 e' 0

  • La derivata di 2f''(a) e' 0 [2f''(a) e' una costante]
    [2f''(a)]'=0

  • La derivata di 6(x-a)f'''(a) e' la derivata di un prodotto fra la variabile 6(x-a) e la costante f'''(a)
    [6(x-a)f'''(a)]' = 6·f'''(a) + 6(x-a)·0 = 6f'''(a)

  • La derivata di 12(x-a)2fIV(a) e' la derivata di un prodotto fra 12(x-a)2 e la costante fIV(a)
    [12(x-a)2fIV(a)]' = 24(x-a)·fIV(a) + 12(x-a)2·0 = 24(x-a)·fIV(a)

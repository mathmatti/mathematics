Serie di funzioni
Una serie di funzioni e' una serie i cui termini sono funzioni
ricordo che una serie e' una successione data dalla somma dei termini di una successione cioe' data la successione
a1, a2, a3, a4, a5, ...... an, .....
e' possibile definire la serie di termini
s1, s2, s3, s4, s5, ...... sn, .....
assegnando ai termini i valori:
s1 = a1
s2 = a1 + a2
s3 = a1 + a2 + a3
s4 = a1 + a2 + a3 + a4
s5 = a1 + a2 + a3 + a4 + a5
..............
..............
sn = a1 + a2 + ...... + an
..............

Sapendo che il primo angolo e' 30° applico la formula
x = (-1)k ° + k 180°
  • per k = 0
    x = (-1)0 30° + 0 · 180° = 30°

  • per k = 1
    x = (-1)1 30° + 1 · 180° = - 30° + 180° = 150°

  • per k = 2
    x = (-1)2 30° + 2 · 180° = + 30° + 360° = 390°   
    (360° + 30°)

  • per k = 2
    x = (-1)3 30° + 3 · 180° = - 30° + 540° = 510°   
    (360° + 150°)

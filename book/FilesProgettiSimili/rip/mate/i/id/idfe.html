Esempio:

nel triangolo ABC ho
a = 5 cm.      b = 4 cm.      c = 3 cm

troviamo un angolo ad esempio

cos = -a2 + b2 + c2
-----------------
2bc 		-(52) + 42 + 32
= ----------------- =
2·(4)(3) 		0
-------
24 		= 0
In questo caso ho ottenuto 0 quindi si tratta di un angolo di 90° (il triangolo di lati 3,4 e 5 e' rettangolo)

Vediamo invece di trovare il valore dell'angolo

cos = a2 - b2 + c2
-----------------
2ac 		52 - 42 + 32
= ----------------- =
2·(5)(3) 		18
-------
30 		= 0,6


quindi, usando le tavole o la calcolatrice ottengo per beta un valore di circa 53°.
se voglio un valore preciso devo scrivere
= arc cos 0,6
Cioe' beta e' l'arco il cui coseno vale 0,6

Proprieta' delle operazioni unione ed intersezione

Vediamo in questa pagina le principali proprieta' dell'unione e dell'intersezione
E' possibile dimostrarle abbastanza facilemente utilizzando i diagrammi di Eulero-Venn, (fare in futuro)
Proprieta' commutativa

per l'unione :      A B = B A

per l'intersezione:      A B = B A
Proprieta' associativa

per l'unione :      (A B) C = A (B C)

per l'intersezione:      (A B) C = A (B C)
Proprieta' di idempotenza

per l'unione :      A A = A

per l'intersezione:      A A = A
Proprieta' dell'insieme vuoto

per l'unione :      A Ø = Ø A = A
per l'unione l'insieme vuoto e' l'elemento neutro

per l'intersezione:      A Ø = Ø A = Ø
per l'intersezione l'insieme vuoto e' l'elemento assorbente
Proprieta' distributiva

dell'unione rispetto all'intersezione:
     A (B C) = (A B) (A C)

dell'intersezione rispetto all'unione:
     A (B C) = (A B) (A C)
Proprieta' di assorbimento

     A (A B) = A

     A (A B) = A

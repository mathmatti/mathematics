1 Definizione

Dato un insieme S, si chiama distanza qualunque funzione/applicazione/relazione che per arbitrari x,y,z ∈ S soddisfa le seguenti proprietà:

  1. d(x,y) >= 0;
  2. d(x,y) = 0 se e solo se x = y;
  3. d(x,y) = d(y,x);
  4. d(x,y) <= d(x,z) + d(z,y).
